To publish through GoDaddy hosting:
1. Sign in to GoDaddy and open My Products.
2. Under Web Hosting, choose Manage and open the File Manager (or use FTP).
3. Open the public_html folder for the daolabz domain.
4. Upload index.html and styles.css from this folder. If a default index file exists, replace it.
5. Visit your domain in a private/incognito browser once the upload completes.

If your GoDaddy account has only the domain, not a Web Hosting plan, tell Codex the exact domain name and preferred host. A free static host can be set up and connected by updating GoDaddy DNS records.
